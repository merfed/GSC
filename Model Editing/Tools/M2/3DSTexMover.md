- **Author**: Tigurius
- **Syntax**: `3DSTexMover.exe <M2 file name> <3ds file name> [<M2to3ds: r>]`
- **AKA**: 3DSToM2Texmover

Copies the UV coordinates from a .3ds file to a M2 file. nUV of 3ds must equal nUV of M2. Option `r` copies the coordinates the other way around.
