- **Author**:
- **Syntax**: `SetChunkFlags.exe <map filename> [<flag: 8>]`

Sets the MCNK flags for all chunks in an ADT.

| Flag | Value |
| ---- | ----- |
| 4    | River |
| 8    | Ocean |
| 16   | Magma |
